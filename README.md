# 📕 RedNote Trends - 小红书热点采集分析工具

一个基于 Web 的小红书热点数据采集与分析工具，支持热门笔记展示、评论分析、类别分布统计和关键词云等功能。

## ✨ 功能特性

- 🔥 **热门笔记排行** - 实时展示热门笔记，支持按类别筛选
- 📊 **类别分布分析** - 可视化展示各类别内容占比
- 💬 **评论情感分析** - 自动分析评论情感倾向（正面/中性/负面）
- 🏷️ **热门关键词云** - 展示当前最热门的话题关键词
- 📈 **24小时热度趋势** - 展示全天热度变化曲线
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 📤 **数据导出** - 支持导出 JSON 格式分析报告

## 🔧 扩展方向

当前版本使用模拟数据，后续可扩展：

1. **接入真实 API** - 对接小红书开放平台或爬虫数据
2. **后端服务** - Python/Node.js 后端实现定时采集
3. **数据存储** - 接入数据库持久化历史数据
4. **AI 分析** - 使用 NLP 进行更精准的情感分析
5. **告警推送** - 热点变化时自动推送通知
